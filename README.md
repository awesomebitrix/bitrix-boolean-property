## Install
```bash
composer require serginhold/bitrix-boolean-property
```
Composer settings to install the module in a local folder:
```json
"extra": {
  "installer-paths": {
    "local/modules/{$name}/": ["type:bitrix-module"]
  }
}
```

## Requirements
* PHP >= 5.5.0
